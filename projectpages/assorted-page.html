Selected p5js Exercises

OBJECTIVE AND INSPIRATION

WHAT:

Build a p5.js sketch that emulates a natural process. This natural process can be any changing process you're interested by –– Wind Motions, Falling Water, Hot Lava, Macaroni Boiling etc

WHY:

To explore methods for emulating natural phenomena in code

PROCESS | CODING

After finding some very helpful tutorials, I learned how to create javascript that would result in a design resembling a sea anenome swaying in the sea.
